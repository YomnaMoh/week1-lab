Describe the relationship between HTML, CSS, and Javascript. Which one(s) are required to create a webpage?

HTML is the base of the site and contains content and text. CSS is the styling portion where you’re able to customize the webpage a little more - including color, font and layout. Javascript is used to add interactive behavior to webpages such as animate images or create dramatic movements with the content.
